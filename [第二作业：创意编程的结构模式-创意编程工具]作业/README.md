上传内容包含两份
=================

hola.pde & Pixel.pde
-----------------------

___hola___
--------------
通过对文字的创意性更改,以你好和侬好为主体,形成一种词云的形式.
在对颜色进行控制和呈现后,随机组合.

***设计展示***

![你好](https://user-images.githubusercontent.com/85673360/141050812-e44769c7-7536-46ee-b12a-7a93fdba9609.png)


___Pixel___
-----------
通过单元圆的规则重复,甚至不规则重复,在经典的旋转构成下,探索形体可能呈现的多种结果,
继而通过简单的重复形成复杂的形体.
