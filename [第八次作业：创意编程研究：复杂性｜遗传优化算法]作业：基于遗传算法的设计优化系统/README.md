以粒子系统为基础的遗传发生行为的优化系统
========================================

第八次作业-基于遗传算法的设计优化系统
--------------------------

***设计说明***
基于粒子系统进行遗传发展,形成粒子系统的演变与优化,进行遗传行为.从而通过遗传行为观察进行衍生发展而形成系统的最优化形式
